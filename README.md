# e-plantShopping

e-plantShopping is a React-based online plant shop application built with Redux Toolkit. Users can browse plants by category, add them to a shopping cart, update quantities, and remove items.

## Project Name
e-plantShopping - Paradise Nursery

## Features
- Browse plants grouped by categories
- Add to Cart with Redux state management
- Update item quantities
- Remove items from cart
- About Us page

## Tech Stack
- React, Redux Toolkit, CSS3
