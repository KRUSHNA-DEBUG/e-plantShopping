# Paradise Nursery 🌿

**Paradise Nursery** is a React-based online plant shop application that allows users to browse a variety of plants, add them to a shopping cart, update quantities, and remove items.

## Project Name
Paradise Nursery

## Features
- Browse a wide selection of plants
- Add plants to your shopping cart
- Update item quantities
- Remove items from the cart
- About Us page with company details

## Tech Stack
- React
- Redux Toolkit
- CSS3

## Getting Started

```bash
npm install
npm start
```
